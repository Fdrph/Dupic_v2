# Dupic  

Simple, minimal similar/duplicate image finder and cleaner that supports PNG, JPG, TIF, WEBP.

Despite being simple, Dupic is faster and easier to use than a lot of commercial software.

This is a complete rewrite in C++, using Clay for the UI layout and SDL3 for the 2D renderer and cross platform stuff. 
Libvips is used for image manipulation and loading.

YOU CAN RIGHT CLICK IMAGES TO SELECT TO DELETE !!!!
